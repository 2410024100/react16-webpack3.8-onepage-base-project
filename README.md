## 项目步骤

1.安装node.js

2.安装项目依赖包

```
npm install
或
yarn install
```

3.运行开发环境

```
npm run dev 

```

4.打包生产文件

```
npm run build
``` 

## flow使用请参考官网：https://flow.org/

```
使用flow检查的文件在顶部增加

npm run flow check --all 


```
